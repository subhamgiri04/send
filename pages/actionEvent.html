actionEvent

Event Manager for actionSpace.

  • constructor(bubble = true)

    bubble, if true will bubble the created events upto the ancestors.

  • addListener(domElement, events, func, ...args)

    domElement - target element
    events - target events (seperated by spaces)
    func - the actual function
    args - optional arguments to func (as an array)
    First argument to the function should be event.

  • addRequestListener(domElement, events, func, args = {})

    domElement - target element
    events - target events (seperated by spaces)
    func - requestModel/name of requestModel
    args - optional arguments to func (as an object)
    The event can be accessed in the requestModel func by l.event

  • removeListener(events)

    Removes all events(seperated by space) listeners in this event manager object.
    removeListener applies to all elements.

  • removeElemListener(domElement, events)

    Removes all events(seperated by space) listeners in this event manager object on a specified domElement.
